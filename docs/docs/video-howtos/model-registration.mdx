# Model Registration Video Tutorials

## Overview

In this video series we walk through registering a model to Dojo using the [MaxHop](https://github.com/jataware/maxhop) locust hopper presence
prediction model as our use case. This model is written in R but most any model can be registered to Dojo as long as it can be run in a Debian
environment. Check out our [model registration](../model-registration) documentation for more detail.
