## Rare Earth Element Mining Scenario Case Study

This is a complex example scenario without commentary. The scenario uses a series of geological
features (things like crust type, geological era, etc.), combined with a land cover type dataset
to identify suitable areas for mining Rare Earth Elements (REEs), taking into account the probability
of REEs being present, and the proximity to bodies of freshwater as water is a key resource in mining operations.
